# 개발일지 | 2026.03.02

## 📌 요약

KUSPBA 프로젝트 초기 세팅부터 메인 기능 구현, 배포 설정까지 1일차 개발을 진행했다.

---

## ✅ 완료된 작업

### 1. 프로젝트 초기 세팅
- Next.js 16 (App Router) + TypeScript + Tailwind CSS v4
- Pretendard Variable 폰트 적용
- KUSPBA 브랜드 컬러 (청록/파랑 계열) 적용
- 폴더 구조 설계 (components, lib, types)

### 2. 공통 컴포넌트
- **Header**: 로고, 네비게이션(협회 소개, 세미나/교육, 학과 제휴), 카카오 로그인 버튼
- **Footer**: 협회명, 저작권
- **로고**: 좌측 상단 로고 이미지 적용
- **디딤이 마스코트**: 협회 소개 페이지 적용

### 3. 메인 페이지
- 히어로 섹션: 슬로건 "우리는 머뭅니다. 당신이 멈추지 않도록."
- 세미나 & 교육 프로그램 섹션 (카드 3개)
- 학과 제휴 배너 (카카오톡 문의 링크 연결)
- 진행 중인 세미나 보기 / 목록 보기 버튼 → `/seminars` 연결

### 4. 협회 소개 페이지 (`/about`)
- Our Message, Core Values(연결/개척/토대), Our Philosophy
- 디딤이 마스코트 섹션
- 메인으로 돌아가기 버튼

### 5. 세미나 관련 페이지
- **세미나 목록** (`/seminars`): 필터 탭(전체/모집 중/마감), 검색, 페이지네이션, type 라벨
- **세미나 상세** (`/seminars/[id]`): 상세 정보, 신청 폼(이름/소속/연락처/이메일/협회원 여부), 무통장 입금 안내
- **신청 완료** (`/seminars/[id]/complete`): 입금 정보, 계좌 복사, 안내 문구
- 메인으로 돌아가기 버튼 추가

### 6. 네비게이션 및 링크
- 네비 "세미나/교육" → `/seminars`
- 히어로 "진행 중인 세미나 보기" → `/seminars`
- 섹션 "진행 중인 세미나 목록 보기" → `/seminars`
- 학과 제휴 카카오톡 링크 → https://pf.kakao.com/_XHhSn

### 7. 배포
- GitHub 저장소 생성 및 push (`baeshh/KUSPBA`)
- Vercel Git 연동으로 자동 배포 설정
- 도메인: kuspba.vercel.app
